# 文件过滤

如图所示，进入文件过滤页面：

<img className={'sm:w-1/2'} src="https://cdn.jun6.net/uPic/2022/08/16/zsxtIk.png"/>

文档内容支持 `markdown` 语法，支持隐藏方式：

- `仅隐藏`：仅隐藏，不限制直接通过 URL 打开，也不影响隐藏内容的直链下载
- `隐藏并不可访问`：隐藏且不可直接通过 URL 打开，也不影响隐藏内容的直链下载
- `隐藏并不可访问不可下载`：隐藏且不可直接通过 URL 打开，隐藏内容不可进行直链下载
