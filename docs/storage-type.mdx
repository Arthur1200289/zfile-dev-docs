## 流量路径

**<font color="green">直传</font>**：表示不额外经过服务器中转

**<font color="red">中转</font>**：表示额外需要在 ZFile 所在服务器对下载、上传进行中转。

**<font color="#679ddd">同 Web 下载，打包行为在浏览器</font>**：打包下载的实现方式为浏览器依次调用每个文件的普通下载，然后通过浏览器将这些文件合成为一个压缩包。（***使用了 Stream API 来支持此功能，所以需要浏览器支持，目前已知 Chrome 的最新版是支持此功能最完善的，其他浏览器可能使用时会有差异或压根不支持。***）
